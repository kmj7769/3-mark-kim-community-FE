# SAY_SOMETHING!

말 못할 고민들을 털어 놓는 회원 기반 커뮤니티 서비스

- 개발 기간: 2025.09 ~ 2025.12
- 개발 인원: 프론트엔드/백엔드/클라우드 (1명)

## 프론트엔드 소개

### 주요 기능

- 회원가입 및 로그인

  - 회원가입
  - 로그인

- 게시글

  - 목록 조회
  - 작성
  - 상세 조회

- 댓글 및 좋아요
  - 댓글 목록 조회
  - 댓글 작성
  - 댓글 수정
  - 댓글 삭제
  - 게시글 좋아요
